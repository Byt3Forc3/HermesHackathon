# Privacy Policy for BrilliantMind

**Effective Date:** March 2026

## Overview
BrilliantMind is a browser extension designed to improve web accessibility and usability for individuals with different cognitive and visual needs. This Privacy Policy explains what information the extension collects, how it is used, and how it is protected.

## Information We Collect

### User Preferences
BrilliantMind allows users to select accessibility settings such as reading mode, simplified page layouts, and other accessibility-related features. These preferences may include selections related to conditions such as dyslexia, ADHD, autism, color blindness, or other accessibility needs.

Currently, these preferences are stored locally in the user’s browser using Chrome's storage API.

### Usage Data (Future Feature)
In future versions of the extension, BrilliantMind may collect anonymized usage patterns to help improve accessibility features. This may include:

- Which accessibility features are enabled
- Interaction patterns with accessibility tools
- General website interaction behavior related to accessibility adjustments

This data will be anonymized and used only for improving the extension.

Users will be informed and given control over this feature before any such data collection occurs.

## How We Use Information
Information collected by BrilliantMind is used to:

- Provide and maintain accessibility features
- Remember user preferences across browsing sessions
- Improve the effectiveness of accessibility tools
- Guide future development of accessibility features

## Data Storage
Currently, all user preferences are stored locally on the user's device using the Chrome extension storage system.

No personal information is transmitted to external servers.

If anonymized analytics are introduced in the future, the storage and processing of such data will follow applicable privacy regulations.

## Data Sharing
BrilliantMind does **not sell, trade, or share personal information with third parties**.

If anonymized analytics are introduced in the future, the collected data will only be used internally for improving the extension.

## User Control
Users maintain full control over their data. They may:

- Change accessibility settings at any time
- Disable the extension
- Remove the extension entirely, which deletes stored preferences

## Security
We take reasonable measures to protect stored information. Since most data is stored locally in the browser, it remains under the control of the user.

## Changes to This Policy
This privacy policy may be updated as the extension evolves. Any updates will be reflected on this page.
