Comprehensive Accessibility Features

🧩

Autism Profile

Reduces sensory overload and creates a calm, predictable browsing environment.

⚡

ADHD Profile

Minimizes distractions to help maintain focus on essential content without being overwhelmed.

📖

Dyslexia Profile

Optimizes typography, spacing, and layouts for easier and more comfortable reading.

⚠️

Epilepsy Profile

Prevents flashing animations and limits hazardous color combinations.

👁️

Color Blindness

Applies a colour shield to make it accesible for protanopia, deuteranopia and tritanopia.

🔍

Partial Blindness

Concentrates the elements of the webpage in the part of the screen that is selected by the user

✨

Simplify Page

A customization feature that allows the user to modify the aspect of the webpage

📚

Reader Mode

An e-reader convertor with an integrated AI-powered assistant for generating summaries, flashcards, and other learning aids.
